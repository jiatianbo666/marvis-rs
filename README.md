## вңЁ ж ёеҝғзү№иүІ

### рҹҸў еҲӣж–°зҡ„"AI еҠһе…¬е®Ө"еҸҜи§ҶеҢ–

иҝҷжҳҜ Marvis жңҖзӢ¬зү№зҡ„и®ҫи®ЎгҖӮе®ғжҠҠе№•еҗҺй»‘зӣ’иҲ¬зҡ„ AI жү§иЎҢиҝҮзЁӢпјҢеҸҳжҲҗдәҶдёҖдёӘ**зңӢеҫ—и§ҒгҖҒж‘ёеҫ—зқҖзҡ„иҷҡжӢҹеҠһе…¬е®Ө**гҖӮ

еҶ…зҪ®дәҶдёҖдёӘз”ұ **6 дёӘ AI Agent** з»„жҲҗзҡ„"иҷҡжӢҹеӣўйҳҹ"пјҢеҗ„еҸёе…¶иҒҢпјҢе№¶иЎҢе·ҘдҪңпјҢе°ұеғҸдёҖдёӘй«ҳж•Ҳзҡ„"ж•°еӯ—йғЁй—Ё"еңЁдёәдҪ жңҚеҠЎпјҡ

- рҹҗ» **Cute Bear**пјҲPM / и°ғеәҰиҖ…пјүвҖ” еҲҶжһҗд»»еҠЎгҖҒеҲҶй…Қе·ҘдҪңгҖҒжұҮжҖ»з»“жһң
- рҹҰ’ **Giraffe**пјҲж–Үд»¶дё“е®¶пјүвҖ” жү«жҸҸж–Үд»¶гҖҒиҜ»еҶҷж“ҚдҪң
- рҹҰҷ **Llama**пјҲзі»з»ҹзӣ‘жҺ§пјүвҖ” CPUгҖҒеҶ…еӯҳгҖҒиҝӣзЁӢз®ЎзҗҶ
- рҹҗ° **Rabbit**пјҲеә”з”Ёз®ЎзҗҶпјүвҖ” жү§иЎҢе‘Ҫд»ӨгҖҒеүӘиҙҙжқҝгҖҒзі»з»ҹж“ҚдҪң
- рҹҗ¶ **Doggie**пјҲжөҸи§ҲеҷЁпјүвҖ” жү“ејҖзҪ‘йЎөгҖҒжҗңзҙўгҖҒжҠ“еҸ–еҶ…е®№
- рҹҗ· **Pig**пјҲжҗңзҙўзӯ–з•ҘпјүвҖ” е…ій”®иҜҚжҸҗеҸ–гҖҒжҗңзҙўи§„еҲ’

жҜҸдёӘ Agent йғҪжңүиҮӘе·ұзҡ„**е·ҘдҪҚ**е’Ң**дәәи®ҫ**пјҢйҖҡиҝҮ Lottie еҠЁз”»з”ҹеҠЁе‘ҲзҺ°гҖӮжІЎд»»еҠЎж—¶пјҢе®ғд»¬дјҡ рҹҳҙ жү“зӣ№гҖҒвҳ• е–қе’–е•ЎгҖҒрҹҸӢпёҸ еҒҘиә«пјӣжҺҘеҲ°д»»еҠЎеҗҺпјҢдҪ дјҡе®һж—¶зңӢеҲ°и°ҒеңЁиҙҹиҙЈгҖҒи°ҒеңЁ"ж‘ёйұј"гҖӮиҝҷз§Қи®ҫи®Ўе°Ҷ **AI жү§иЎҢиҝҮзЁӢе®Ңе…ЁйҖҸжҳҺеҢ–**пјҢжһҒеӨ§зј“и§ЈдәҶз”ЁжҲ·еҜ№ AI"й»‘з®ұ"зҡ„дёҚдҝЎд»»ж„ҹпјҢи®© AI д»ҺзҘһз§ҳе·Ҙе…·еҸҳжҲҗдәҶ**еҸҜз®ЎзҗҶзҡ„"ж•°еӯ—еҠіеҠЁеҠӣ"**гҖӮ

### рҹ”§ зңҹе®һж“ҚжҺ§з”өи„‘

иҫ“е…ҘиҮӘз„¶иҜӯиЁҖ вҶ’ AI еҲҶжһҗ вҶ’ и°ғеәҰ Agent вҶ’ **зңҹе®һжү§иЎҢзі»з»ҹе·Ҙе…·**гҖӮдёҚеҸӘжҳҜеҜ№иҜқпјҢжҳҜзңҹзҡ„еҠЁжүӢгҖӮ

## йЎ№зӣ®иғҢжҷҜ

жң¬йЎ№зӣ®дёә Rust иҜҫзЁӢжңҹжң«еӨ§дҪңдёҡпјҢеұ•зӨә Rust е·ҘзЁӢеҢ–ејҖеҸ‘зҡ„е®Ңж•ҙиғҪеҠӣпјҡ
- Workspace жЁЎеқ—еҢ–жһ¶жһ„пјҲ7 дёӘ crateпјү
- жүҖжңүжқғ/еҖҹз”ЁгҖҒTrait/жіӣеһӢ/з”ҹе‘Ҫе‘Ёжңҹзҡ„еҗҲзҗҶдҪҝз”Ё
- ејӮжӯҘзј–зЁӢпјҲTokioпјүгҖҒй”ҷиҜҜеӨ„зҗҶпјҲanyhow + thiserrorпјү
- еҚ•е…ғжөӢиҜ•иҰҶзӣ–пјҲ119 дёӘжөӢиҜ•пјү
- Tauri жЎҢйқўеә”з”Ёжү“еҢ…

## еҝ«йҖҹејҖе§Ӣ

### зҺҜеўғиҰҒжұӮ

| е·Ҙе…· | зүҲжң¬иҰҒжұӮ | иҜҙжҳҺ |
|------|---------|------|
| Rust | 1.75+пјҲжҺЁиҚҗ 1.96+пјү| `rustup` е®үиЈ… |
| Node.js | 18+ | еүҚз«Ҝжһ„е»ә |
| npm | 9+ | еҢ…з®ЎзҗҶ |
| Windows | 10/11 | жЎҢйқўеә”з”Ёе№іеҸ° |

### зј–иҜ‘иҝҗиЎҢпјҲжЎҢйқў Appпјү

```bash
# 1. е…ӢйҡҶйЎ№зӣ®
git clone <repo-url>
cd rust-final

# 2. е®үиЈ…еүҚз«Ҝдҫқиө–
cd marvis-app
npm install

# 3. й…ҚзҪ® API Key
# зј–иҫ‘ marvis-app/src-tauri/src/commands.rs дёӯзҡ„ DEEPSEEK_KEY
# жҲ–иҖ…еңЁйЎ№зӣ®ж №зӣ®еҪ•еҲӣе»ә .env ж–Үд»¶

# 4. жһ„е»әжЎҢйқўеә”з”Ё
npx tauri build

# 5. иҝҗиЎҢ
./target/x86_64-pc-windows-gnu/release/marvis-tauri.exe
```

### зј–иҜ‘иҝҗиЎҢпјҲCLI е·Ҙе…·пјү

```bash
# жһ„е»ә CLI зүҲжң¬
cargo build --release -p marvis-cli

# дҪҝз”Ё Mock AIпјҲж— йңҖ API Keyпјү
cargo run --release -p marvis-cli -- repl

# дҪҝз”Ё DeepSeek зңҹе®һ AI
cargo run --release -p marvis-cli -- -P deepseek -m deepseek-v4-pro repl

# еҚ•ж¬Ўжү§иЎҢ
cargo run --release -p marvis-cli -- run "жҹҘзңӢзі»з»ҹдҝЎжҒҜ"
```

## й”ҷиҜҜеӨ„зҗҶ

йЎ№зӣ®дёҘж јйҒөеҫӘ Rust й”ҷиҜҜеӨ„зҗҶжңҖдҪіе®һи·өпјҡ

- вқҢ **зҰҒжӯў** `unwrap()` / `expect()`пјҲд»…жөӢиҜ•е…Ғи®ёпјү
- вң… `Result<T, MarvisError>` з»ҹдёҖй”ҷиҜҜзұ»еһӢ
- вң… `?` ж“ҚдҪңз¬Ұдј ж’ӯй”ҷиҜҜ
- вң… `anyhow::Context` ж·»еҠ дёҠдёӢж–ҮдҝЎжҒҜ
